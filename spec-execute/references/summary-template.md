## Frontmatter 字段说明

| 字段 | 必填 | 说明 | 可选值 |
|------|------|------|--------|
| `title` | 是 | 文档标题 | - |
| `type` | 是 | 文档类型 | `summary` |
| `category` | 是 | 分类目录 | 与 plan.md 相同 |
| `status` | 是 | 当前状态 | `未确认`/`已确认`/`已归档` |
| `created` | 是 | 创建日期 | `YYYY-MM-DD` 格式 |
| `plan` | 是 | 关联的 plan | `"[[plan]]"` |
| `tags` | 是 | 标签列表 | 至少包含 `spec` 和 `summary` |

## status 状态变更规则

| 触发条件 | 状态变更 | 说明 |
|----------|----------|------|
| 创建 summary.md | → `未确认` | 初始状态 |
| 用户确认内容 | `未确认` → `已确认` | 用户确认无误 |
| 用户修改文档 | `已确认` → `未确认` | 需重新确认 |
| 归档完成 | `已确认` → `已归档` | 归档时更新 |
