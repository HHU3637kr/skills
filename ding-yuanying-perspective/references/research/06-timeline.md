# 丁元英 · 人物时间线

> 注：丁元英为虚构人物，以下时间线基于小说设定

## 关键节点

| 时间 | 事件 | 对思维的影响 |
|------|------|-------------|
| 1959年 | 出生于成都 | — |
| 1978年 | 考入清华大学 | 系统性思维的基础训练 |
| 1979年 | 留学柏林洪堡大学 | 接触西方经济学和哲学体系，形成跨文化视角 |
| 1985年 | 获经济学硕士 | 经济学成为分析工具 |
| 1985-1989年 | 就职柏林H.N.S国际金融投资公司 | 实战金融，理解资本运作底层逻辑 |
| 1989年 | 就职北京通达证券公司 | 回国，看到中国股市的制度漏洞 |
| 1990年 | 结婚同年离婚 | 对世俗关系的疏离感加深 |
| 1991年 | "鬼混"期 | 思想沉淀期 |
| 1992年 | 就职柏林《世界经济周刊》 | 宏观经济视角 |
| 1994年1月 | 辞职 | 决定独立运作 |
| 1994年6月 | 创办私募基金 | "文化密码"实战——破译中国股市的文化属性 |
| 1995年5月 | 解散私募基金 | 良心觉醒——"不是打仗，是屠杀"；资金被冻结，5年禁入股市 |
| 1995-1998年 | 古城隐居 | 简朴生活，吃方便面，听音响，思考 |
| 1996年 | 认识芮小丹 | 遇到"天国的女人"——唯一理解他的人 |
| 1997年 | 王庙村扶贫/格律诗项目 | 文化属性论的实践验证——"杀富济贫" |
| 1997年 | 五台山论道 | 为"杀富济贫"讨心安，与智玄大师论道 |
| 1998年 | 芮小丹牺牲 | 精神世界的崩塌——唯一能"站着"和他对话的人消失了 |
| 1998年 | 格律诗胜诉/乐圣合作 | 王庙村脱贫，商业布局完成，但代价沉重 |

## 思想转折点
1. **私募基金→解散**：从"破译文化密码赚钱"到"利用愚昧赚钱的道德觉醒"
2. **隐居→王庙村**：从旁观到入局，但以"不入局"的方式入局
3. **芮小丹之死**：理性无法解释的失去，对"天道"的终极叩问

## 虚构人物的特殊性
- 无"最新动态"，时间线止于小说结尾
- 人物形象由作者豆豆塑造，反映的是作者的思想而非真实人物
- 电视剧（2008年）对小说有删减和改编，部分情节有差异

## 来源
- 小说设定（豆瓣、百科等汇总）
- 可信度：一手（小说设定）
