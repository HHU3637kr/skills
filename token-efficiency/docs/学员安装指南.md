# 学员安装指南

**token-efficiency v0.2.0** — 独立省 Token 插件（单独安装，单独使用）

支持 **Cursor · Codex · Claude Code · Hermes · Windsurf** 等。

---

## 最简单：把 zip 给 Agent 装（推荐）

1. 解压 `token-efficiency-0.2.0.zip` 到任意位置（如桌面）
2. 打开 Cursor / Codex / Claude Code
3. **复制下面发给 Agent**（改路径）：

```
我解压了 token-efficiency 插件，路径是：/Users/你的用户名/Desktop/token-efficiency-0.2.0

请阅读该目录的 AGENT_INSTALL.md，帮我：
1. 运行 install.sh（Windows 用 install.ps1）
2. 运行 python3 scripts/setup_student.py
3. 把 audit 报告 Summary 发给我

Hermes 用户：安装完成后在 TUI 输入 /reload-skills
```

Agent 会帮你完成安装，无需手敲命令。

---

## 手动安装

```bash
cd token-efficiency-0.2.0
bash install.sh
python3 scripts/setup_student.py
```

Windows：`.\install.ps1` 然后 `python scripts\setup_student.py`

---

## 装完后

对 Agent 说：**「token audit」** 或 **「省 token」**

## 常见问题

**没有 python3？** Windows 装 Python 3 并勾选 PATH，用 `python` 代替。

**会改坏 skills 吗？** `fix_skills` 会建 `.bak` 备份；用 `--keep` 保留常用 skill。

**和其他工具冲突吗？** 不冲突；规则带 `token-efficiency-` 前缀，默认不覆盖已有文件。
