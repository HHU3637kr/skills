# token-efficiency 内测公告

各位好，发一个**独立的省 Token 插件**给大家测试。

**名字：token-efficiency**  
**性质：独立 Skill + 工具包**（不是其他任何产品的组件，单独安装、单独用）

---

## 一、它能帮你省多少？（诚实区间）

整体：**重度用户常见可降 40%–70%**（固定税 + Agent 行为双降）  
轻量用户：**20%–40%**（主要看 Agent 使用习惯）

下面按「省在哪里」拆开说——**每一项都有原因，不是 magic prompt**。

---

## 二、每一块能省多少？为什么能省？

### ① L0 固定税 — 每轮对话都白付的钱（复利最大）

| 浪费点 | 典型浪费 | 修复方式 | 预计节省 | 原因 |
|--------|----------|----------|----------|------|
| 几十个 Skill 描述常驻 system | 1,000–3,000 tok/**每轮** | `fix_skills` 改按需加载 | **40%–60% 固定税** | API 无状态，每 turn 重发 skill 描述 |
| MCP 开太多 server | 2,000–15,000 tok/**每轮** | 关掉不用的 MCP | **按 server 数线性降** | 每个 server 的 tool schema 每轮都带上 |
| CLAUDE.md / rules 太臃肿 | 500–5,000 tok/**每轮** | `compress` + 删废话 | **20%–50% 该文件** | always-on 文件每轮计费 |
| alwaysApply 大规则 | 500–5,000 tok/**每轮** | 改 glob 触发 | **只在相关文件时加载** | 不必 Python 项目也加载 React 规则 |

**一句话：** 你装越多「常驻配置」，每说一句话都在重复交钱。

---

### ② L1 Agent 乱探索 — 任务 token 的最大头

| 浪费点 | 典型浪费 | 修复方式 | 预计节省 | 原因 |
|--------|----------|----------|----------|------|
| 重复读同一文件 | 每多读一次 = 整文件 token | behavior 规则：session 内不重复读 | **30%–50% 任务 token** | tool 返回全进 context |
| 「先看看仓库」式 explore | 5,000–10,000 tok 还没开工 | grep 优先、禁止盲目 explore | **一大截前置成本** | 10 个文件 speculative read |
| 串行 tool call | 多一轮 = 多付一轮固定税 | 批并行 | **turn 数直接减少** | 每 turn 重计固定税 |

**实测参考**（开源 Minimize-Cursor-Cost，20 任务）：Agent 定位+改一处 **7,000 → 2,200 tok（-69%）**。

---

### ③ L3 输出废话 — 回答太长

| 浪费点 | 典型浪费 | 修复方式 | 预计节省 | 原因 |
|--------|----------|----------|----------|------|
| 「Sure! Let me…」+ 改完又总结 | 输出 40%–70% 可删 | output 规则 | **输出 token -40%–70%** | 模型默认讨好式长回复 |
| 整文件 rewrite | 2,000 tok vs patch 200 tok | 只出 diff | **90% 该次输出** | 同样改动 patch 更小 |

---

### ④ L4 / L5 — 会话 & 模型档

| 浪费点 | 修复 | 预计节省 | 原因 |
|--------|------|----------|------|
| 一个 chat 聊 10 个话题 | 新开会话 | 简单问答 **60%–90%** | 历史全带上 |
| 问路径用小问题却开 Agent | 用 Ask 模式 | 无 tool 开销 | Agent 模式固定更重 |
| 小活用 Opus/deep | 探索用 fast，写代码再升档 | 单次 **5×–20×** | 模型单价差 |

## 三、怎么安装？（最简单）

### 方式 A：把 zip 丢给 Agent（推荐）

1. 解压 `token-efficiency-0.2.0.zip`
2. 对 Agent 说：

```
我解压了 token-efficiency 插件，路径是：【你的解压路径】
请读 AGENT_INSTALL.md，帮我安装并跑 setup_student.py，把 audit 结果摘要发我。
```

Agent 会帮你装完，**不用自己敲命令**。

### 方式 B：自己 3 行（Mac/Linux）

```bash
cd token-efficiency-0.2.0
bash install.sh
python3 scripts/setup_student.py
```

Windows：`install.ps1` + `python scripts\setup_student.py`

## 四、装完怎么用

- **「省 token」**
- **「save tokens」**

它会扫描你的配置，告诉你**先改哪几项 ROI 最高**。
