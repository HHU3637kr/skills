# token-efficiency 产品说明

## 是什么

**token-efficiency** 是**独立的**跨平台、跨 Agent 省 Token 插件（Skill + CLI）。

```
审计固定税 → 按 ROI 修复 → 安装规则 → 再审计验证
```

**独立分发、独立安装、独立使用。** 与其他任何产品无绑定关系。

## 解决什么问题

| 痛点 | 做法 |
|------|------|
| 还没干活 token 没了 | audit 扫 MCP、skills、rules |
| Agent 乱读文件 | behavior 规则 |
| 回答太长 | output 规则 |
| 配置越堆越大 | compress + fix_skills |

## 分发方式

讲师运行 `python3 scripts/package_release.py` → 得到 zip → 学员解压 → Agent 按 `AGENT_INSTALL.md` 安装。

## 版本

**v0.2.0** — 完整可测版：zip 分发、Agent 安装、公告文档、setup_student
